###### Markdown支援有序清單和無序清單。

無序清單使用星號、加號或是減號作為清單標記：

```
* Red
* Green
* Blue
```

* Red
* Green
* Blue
---

使用+號與-號，效果都一樣
```
+ Red
+ Green
+ Blue
- Red
- Green
- Blue
```
+ Red
+ Green
+ Blue
- Red
- Green
- Blue
